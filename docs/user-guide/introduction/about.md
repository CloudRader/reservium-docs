# What is Reservium? :material-help-circle-outline:{ .main-color }

**Reservium** is a modern, open-source reservation system designed to streamline the management of shared spaces within any organization.

---

## :material-office-building-marker: Core Purpose

Reservium helps organizations transition from chaotic manual booking methods (like spreadsheets or emails) to a structured, transparent, and consistent digital process. It is ideal for managing:

- :material-door-open: **Meeting Rooms** and conference halls.
- :material-desk: **Work Areas** and shared desks.
- :material-coffee: **Break Areas** or recreation zones.
- :material-car: **Parking Spaces** or other physical assets.

---

## :material-lightning-bolt-outline: Key Capabilities

### For Users
- **Real-time Availability**: Instantly see which spaces are free via a visual calendar.
- **Easy Booking**: Create reservations with just a few clicks, respecting predefined rules.
- **Automated Notifications**: Receive instant email updates about your booking status.
- **SSO Integration**: Secure login using your existing organization accounts (Keycloak).

### For Managers
- **Flexible Rules**: Define booking windows, capacity limits, and access permissions.
- **Approval Workflows**: Set specific rooms to require manual confirmation.
- **Centralized Panel**: Manage all services, users, and reservations from a single interface.
- **External Sync**: Integration with tools like Google Calendar.

---

## :material-rocket-launch-outline: The Future of Reservium

Reservium is actively developed with a focus on:
- **Advanced Permissions**: Granular multi-tenant features for large organizations.
- **Access Control**: Direct integration with smart locks and IoT hardware.
- **Enhanced Analytics**: Better insights into space utilization and demand.
