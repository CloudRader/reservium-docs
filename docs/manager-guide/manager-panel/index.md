# Manager Panel :fontawesome-solid-sliders:{ .main-color }

The **Manager Panel** is where you configure the structural elements of Reservium. Use these sections to define your rooms, their availability, and any additional services they offer.

---

<div class="grid cards" markdown>

-   :fontawesome-solid-building-circle-check:{ .main-color } __[Reservation Services]__

    ---

    Manage the high-level entities (rooms, spaces) available for booking.

-   :fontawesome-solid-calendar-days:{ .main-color } __[Manage Calendars]__

    ---

    Define when and how specific areas within a service can be reserved.

-   :fontawesome-solid-boxes-stacked:{ .main-color } __[Mini Services]__

    ---

    Configure optional add-ons like equipment or catering for your services.

</div>
